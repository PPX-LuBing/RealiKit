# Reality Scanner TUI

一键扫描 + 深度验证 Reality 协议可用域名的终端 UI 工具。

基于 [RealiTLScanner](https://github.com/XTLS/RealiTLScanner)（TLS 扫描）和 [RealityChecker](https://github.com/V2RaySSR/RealityChecker)（深度验证）。

## 快速开始

```bash
./reality-tui
```

## 扫描方案（按 m 切换）

| 方案 | 线程 | 超时 | 验证 | 适用场景 |
|------|------|------|------|----------|
| **标准** | 10 | 5s | ✅ 深度验证 | 日常使用（默认） |
| **快速** | 20 | 5s | ❌ 仅 TLS | 快速筛查大批量 IP |
| **深度** | 5 | 10s | ✅ 深度验证 | 对少量目标精准检测 |
| **批量** | 50 | 3s | ❌ 仅 TLS | 海量 CIDR 快速扫描 |

## 输入方式

- **IP/CIDR/域名** — `1.2.3.4` / `1.2.3.0/24` / `example.com`
- **从文件** — 每行一个目标，按 `c` 直接验证 CSV（跳过 TLS 扫描）
- **从 URL** — 自动提取页面中的域名

## 参数设置

| 参数 | 说明 |
|------|------|
| 端口 | 默认 443 |
| 线程 | 方案预设，可手动修改 |
| 超时 | 每个目标的最大等待时间 |
| 上限 | 最多扫描多少个目标（0=不限） |

## 快捷键

| 按键 | 功能 |
|------|------|
| `s` | 开始扫描 |
| `m` | 切换扫描方案 |
| `↑` `↓` | 切换输入方式 / 移动结果行 |
| `i` | 切换 IPv6 |
| `v` | 切换详细日志 |
| `c` | 直接验证 CSV（文件模式） |
| `enter` | 查看详情 |
| `1` | 按评分排序 |
| `2` | 按延迟排序 |
| `f` | 显示/隐藏不可用结果 |
| `e` | 导出 CSV |
| `esc` | 返回上级 |
| `q` | 退出 |

## 推荐工作流

1. 选 **标准** 方案（默认）
2. 输入 VPS 的 IP 或 CIDR
3. 按 `s` 开始
4. TLS 扫描 → 自动深度验证 → 5 星评分
5. 选中行按 `enter` 看详情，按 `e` 导出 CSV

## 构建

```bash
go build -o reality-tui ./cmd/reality-tui/
```

需要 Go 1.22+。
