# RealiKit

一键扫描 + 深度验证 Reality 协议可用域名的终端 UI 工具。

集成 [RealiTLScanner](https://github.com/XTLS/RealiTLScanner)（TLS 扫描引擎）和 [RealityChecker](https://github.com/V2RaySSR/RealityChecker)（深度验证引擎），提供傻瓜式 TUI 操作。

## 快速开始

```bash
# 下载或自行编译
./realikit
```

## 特性

### 扫描方案（按 `m` 切换）

| 方案 | 线程 | 超时 | 深度验证 | 场景 |
|------|------|------|----------|------|
| **标准** | 10 | 5s | ✅ | 日常使用（默认） |
| **快速** | 20 | 5s | ❌ | 大批量快速筛查 |
| **深度** | 5 | 10s | ✅ | 少量目标精准检测 |
| **批量** | 50 | 3s | ❌ | 海量 CIDR 扫一圈 |

### 输入方式

- **IP/CIDR/域名** — `1.2.3.4` / `1.2.3.0/24` / `example.com`
- **从文件** — 每行一个目标，按 `c` 直接验证 CSV（跳过 TLS 扫描）
- **从 URL** — 自动提取页面中的域名

### 多维度检测

| 检测项 | 说明 |
|--------|------|
| TLS 协议 | TLS 1.3、h2 ALPN、X25519 |
| HTTP 可达性 | 状态码分类（安全/排除/网络不可达） |
| CDN 检测 | 三级置信度（高/中/低），从 CNAME、HTTP 头、证书签发者检测 |
| 证书验证 | 有效期、SNI 匹配、颁发者 |
| 被墙检测 | GFW 特征关键词匹配 |
| 地理位置 | GeoIP 国家代码 |
| 热门网站 | 内置热门站列表，降低被封风险 |

## 操作

### 快捷键

| 按键 | 功能 |
|------|------|
| `s` | 开始扫描 |
| `m` | 切换扫描方案 |
| `↑` `↓` | 切换输入方式 / 移动结果行 |
| `i` | 切换 IPv6 |
| `v` | 切换详细日志 |
| `c` | 直接验证 CSV（文件模式） |
| `t` | 手动验证模式（输入域名看评分） |
| `S` | 保存当前参数到本地配置 |
| `enter` | 查看详情 / 确认 |
| `space` | 收藏标记 |
| `d` | 域名去重合并 |
| `/` | 搜索过滤 |
| `1` `2` | 按评分 / 延迟排序 |
| `3` `4` `5` | 按最低星级筛选（按 `0` 清除） |
| `f` | 显示/隐藏不可用结果 |
| `F` | 只看收藏 |
| `g` | 生成 Reality 配置（含 x25519 密钥） |
| `e` | 导出 CSV |
| `E` | 仅导出收藏结果 |
| `w` | 导出 HTML 报告 |
| `esc` | 返回上级 |
| `q` | 退出（有数据时二次确认） |

## 推荐工作流

1. 选 **标准** 方案
2. 输入 VPS 的 IP 或 CIDR（如 `1.2.3.0/24`）
3. 按 `s` 开始
4. TLS 扫描 → 自动深度验证 → 5 星评分
5. 最低星级筛选（`4` 或 `5`）
6. 选中域名按 `enter` 看详情，按 `g` 生成 Reality 配置

## 构建

```bash
go build -o realikit ./cmd/reality-tui/
```

需要 Go 1.22+。
